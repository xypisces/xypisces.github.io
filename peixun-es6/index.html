Dva中的ES6知识点

const 与 let

目的：创建块级作用域，let声明变量，const声明常量

let与var用法类似，但是let声明的变量只在所处的代码块中有效

{
    let a = 1;
    var b = 2;
}
console.log(a) //报错
console.log(b) // 2

再看一道经典面试题，下面代码会输出什么？

for (var i=1; i<=5; i++) {
    setTimeout( function timer() {
        console.log(i);
    }, i*1000 );
}

for循环并不会生成一个作用域，每次循环是跟随变量所在的作用域，而var声明的是全局变量，因此for每次循环中访问的i都是全局中同一个i变量

闭包解决方案

for (var i=1; i<=5; i++) {
    (function(i) {
        setTimeout( function timer() {
            console.log(i);
        }, i*1000 );
    })(i)
}

let解决方案

for (let i=1; i<=5; i++) {
    setTimeout( function timer() {
        console.log(i);
    }, i*1000 );
}

使用let的时候，每次循环i都是let声明的一个新变量，每次循环都会有类似代码块包起来执行，本质上达到了立即执行函数的作用

const声明的变量，需要赋值且不可以修改，不可以重复声明。

const obj = {
    a: 1,
    b: 2,
}
obj.a = 2
console.log(obj)

const所谓的不可以修改值，指的是改变量的地址不会被修改。简单数据类型的值指向的就是地址，而对于对象和数组，只能保证他的内存地址指针不变。

箭头函数

目的：简化匿名函数，让this指向固定化。

const f = function(v) {
    return v+1
}
// 等同于
const f = (v) => {
    return v+1
}
// 等同于
const f = v => v+1

箭头函数本身没有this,箭头函数中的this绑定在定义时所在的作用域而是引用外层的this

var obj = {
    a: 2,
    f: function() {
        console.log(this.a)
    }
}
obj.f()

var obj = {
    a: 2,
    f: () => {
        console.log(this.a)
    },
}
obj.f()

generators

用途：异步编程解决方案，用同步写异步

//fetch
getJson = (url) => {
    return fetch(url)
  .then(res => res.json())
  .then(data => ({ data }))
  .catch(err => ({ err }));
}
getJson('/api/xxxxx').then((data) => {
    console.log(data)
}, (err) => {
    console.log(err)
})
//generator
function* request() {
  try {
    let data = yield fetch('/api/xxxxx');
    console.log(data);
  } catch (error) {
    console.log('Oops, error: ', error);
  }
}
